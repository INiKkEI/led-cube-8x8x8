This project is a portfolio-focused 8×8×8 LED cube built as a complete embedded-systems engineering case study rather than just a visual LED demo. The system is centered around a custom ESP32-based control PCB designed to drive 512 LEDs through a multiplexed scanning architecture, supported by dedicated firmware for animation control and system operation. The repository is structured to document the full engineering process, including project overview, hardware design, firmware development, validation, and supporting media, so the final result shows not only the working device but also the reasoning and development process behind it.

The main goal of the project is to create a strong public portfolio piece that demonstrates system-level design, custom PCB development, hardware-software integration, bring-up, troubleshooting, verification, and professional documentation. In that sense, the LED cube serves both as a functional electronics build and as a transparent showcase of engineering workflow, from concept and architecture decisions to implementation and final presentation.
