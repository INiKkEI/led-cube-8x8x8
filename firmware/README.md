# Firmware Workspace

This folder is the workspace for the real firmware project used to run the LED Cube.

It is expected to hold items such as:

- source code
- headers
- build configuration
- test or diagnostic firmware
- supporting project files

This folder is different from `docs/03_firmware/`:

- `firmware/` = actual implementation files
- `docs/03_firmware/` = explanatory documentation about firmware architecture, state flow, BLE behavior, and animation logic

Until the firmware project files are added, this README defines the intended role of the folder.
