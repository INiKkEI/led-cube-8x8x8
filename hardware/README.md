# Hardware Workspace

This folder is the workspace for the physical hardware design of the LED Cube project.

It is expected to hold the real design outputs such as:

- schematic source files
- PCB layout files
- fabrication outputs
- assembly outputs
- library assets if they are version-controlled in this repository

This folder is different from `docs/02_hardware/`:

- `hardware/` = actual design workspace and exports
- `docs/02_hardware/` = explanatory engineering documentation about the hardware design

Until the hardware design files are added, this README exists only to define the intended role of the folder.
